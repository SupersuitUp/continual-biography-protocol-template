# captures/

Raw session records. One dated file per session: `YYYY-MM-DD-<topic-slug>.md`, frontmatter per `AGENTS.md`. Verbatim, unpolished, never edited after landing. Corrections happen in the layers with the contradiction noted.

Modes: `interview` (prepared session), `self-capture` (subject's own chat-app or solo session), `voice-memo` (unprompted recording), `found-source` (a recorded artifact that behaves like a session, e.g. a speech or podcast appearance).

A capture is debt until its `processed:` flag is true.
