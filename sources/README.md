# sources/

Pre-existing raw material: manuscripts, journal scans and transcriptions, letters, clippings, prior interviews. Intake, never homework: file it here first, then process each source through the same Process phase as a capture (see `AGENTS.md`), largest first. Track what remains unprocessed in the `STATE.md` intake queue.

Name files with whatever date is knowable: the material's own date if it has one (`1988-journal-volume-2.md`), the intake date if not (`2026-01-15-manuscript-draft.pdf`).
