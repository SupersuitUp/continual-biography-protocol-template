# Media index

Every photo, video, and audio artifact in the library, one row each. The library grows by asks: every processing pass emits media asks into `STATE.md`, and prep puts them in front of the interviewer.

Small files live in `photos/`, `video/`, `audio/`. Files over ~50MB live in external storage (a private drive folder) with the link in the Location column; git is the index of record either way.
