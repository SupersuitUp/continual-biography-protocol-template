# themes/

One file per through-line of the life: `<theme-slug>.md`. Themes are discovered, never imposed: a through-line earns its own file after roughly three captures touch it. Each file states the theme in one line, then accumulates evidence bullets (dated, source-linked) with every processing pass.

Themes are where projections get their shape: a memoir's chapters, a film's motifs, and an art series' subjects usually fall out of this folder.
