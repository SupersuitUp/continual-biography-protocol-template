# people/

One file per recurring person in the subject's life: `firstname-lastname.md` (or the name the subject uses, kebab-case). First mention in a capture gets a stub; every later mention grows the file. This is the subject's own relationship graph, and it is where an interviewer finds the follow-up question ("you have mentioned your brother in four sessions and never said how he died").

Each file: who they are to the subject, key facts with source links, and open questions about them.
